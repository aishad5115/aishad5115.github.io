Python Project

Data Analysis · Visualization

For this project I used Python to analyze and visualize VO₂ max data. VO₂ max is basically a measure of how much oxygen your body can use during exercise and its a big deal in kinesiology. The notebook uses Pandas and Matplotlib to work with the data and create plots.

VO₂ Max Data Analysis

A Jupyter notebook that looks at VO₂ max values and generates plots from the data. Done in Python as part of KNES 381.

VO₂ Max Calculator

Interactive Tool · Kinesiology

This is an interactive VO₂ max calculator based on the Rockport Walk Test. You enter your info and it estimates your VO₂ max and tells you where you fall in terms of fitness level.
