Scratch Maze Project

This is a 3 level maze game built in MIT Scratch where you control a mouse trying to find the cheese using the arrow keys. It has a countdown timer and a lives system so if you run out of either you lose. Each level gets harder with more complex mazes as you progress. There's also an element of added cheat resistance so the sprite can't be dragged and gets sent back to start if it touches a wall. It covers the core programming concepts from class including variables, loops, conditionals and events.

Python Project

This is a project that uses Python in a Kaggle notebook to analyze VO₂ data from an exercise test. VO₂ measures how much oxygen your body uses during exercise and is a key indicator of aerobic fitness. The notebook generates several plots including a main VO₂ plot and gas exchange graphs. It also marks the Gas Exchange Threshold and Respiratory Compensation Point as reference lines. The script is written to be reusable so it can run on a new dataset just by changing the file path.

GIMP Photo Editing Project

This is a photo montage made in GIMP using at least four layered images combined into one scene. It shows a woman running from an alien on a city boardwalk at night with a galaxy and planet sky in the background. I used layer masking and blending to merge the images together so they sit in the same scene. The goal wasn't realism, just to practice the editing tools and techniques we learned in class.
